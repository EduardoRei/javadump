Atividade
Crie um método estático que recebe uma matriz e que faça a rotação de acordo com o angulo informado.

Por exemplo:

matriz = { { 2, 3}, { 4, 5} } 
rotacionar 90graus
resultado 
matriz = { { 4,2}, {5,3}}

Graus permitidos
0, 90, 180, 270 e 360

Rotacione sempre no sentido horário. 

Considere apenas matrizes quadradas.
